# Showcase

引退OSは、案件選別・受託実績・自社事業化を小さく試すための開発環境です。
このページでは、外部に見せられる成果物を短く紹介します。


## Retirement OS Launcher

既存ツールへアクセスできる GitHub Pages 用のトップページです。
案件選別、応募文作成、案件管理、作業記録、実績化までの流れを1画面で案内します。

### What It Solves

- どのツールから開けばよいか迷わない。
- GitHub Pages 公開時の入口になる。
- ツール集としての全体像を見せやすくする。

### Portfolio Value

単体ツールの集まりではなく、受託活動を支える一連のプロダクトとして見せられます。

## Opportunity Scorecard

案件や企画を同じ基準で採点し、応募 / 保留 / 見送りを判断するブラウザツールです。
HTML ファイルを開くだけで使えます。

### What It Solves

- 受託案件に応募するか迷う。
- ポートフォリオに載せる制作物を選びたい。
- 自社事業アイデアの優先順位を決めたい。
- 判断理由を Markdown で残したい。

### Evaluation Points

- 売上性
- 再利用性
- 実行速度
- 証明価値
- 戦略性
- リスク低さ
- 作業負荷の低さ

### Portfolio Value

このツールは、単なるメモではなく「判断プロセスを資産化する」ための道具です。
受託、制作実績、自社事業のどれにも転用できます。



## Website Improvement Analyzer

企業ホームページ、LP、ECサイトを手動で診断し、改善レポートと営業提案書のたたき台を作るブラウザツールです。

### What It Solves

- 営業前にサイト改善ポイントを整理したい。
- SEO、CTA、スマホ対応、ファーストビュー、信頼材料、表示速度の診断漏れを防ぎたい。
- 提案書や応募文に使える短い版・詳しい版のMarkdownレポートを作りたい。

### Portfolio Value

受託営業、ポートフォリオ、自社事業のすべてに転用できる営業支援ツールです。
将来はURL診断やAI改善提案へ拡張できます。

## Website Fix Plan Builder

Website Improvement Analyzer の診断結果を、改善作業、納品物、作業手順、見積もり目安、クライアント確認事項へ変換するブラウザツールです。

### What It Solves

- サイト診断を提案だけで終わらせず、受託案件として実行できる形にする。
- 小規模事業者向けの改善メニュー、対応範囲、納品物を整理する。
- Work Log Builder と Portfolio Case Builder に転記しやすいMarkdownを作る。

### Portfolio Value

診断、改善計画、提案、管理、作業記録、実績化までの流れをつなぐ中継ツールです。
小規模サイト改善支援を商品化するための実務資産として見せられます。
## Proposal Draft Builder

Opportunity Scorecard で「応募」と判断した案件から、応募文の下書きを作るブラウザツールです。
クラウドワークス等へ貼りやすい Markdown 形式で出力できます。

### What It Solves

- 応募文を毎回ゼロから書く時間を減らす。
- 案件内容、自分の強み、実績を整理して伝える。
- 送信前に調整しやすい下書きを作る。

### Portfolio Value

案件選別から応募文作成までの流れを、実務ツールとしてつなげられます。
受託の営業活動そのものを改善する成果物として見せられます。


## Portfolio Case Builder

完了した案件や制作物を、GitHub README・Notion・ポートフォリオへ貼れる実績紹介文に変換するブラウザツールです。

### What It Solves

- 完了した作業を「やっただけ」で終わらせない。
- 課題、対応内容、工夫、成果を短く整理する。
- 受託実績や自作ツールを公開しやすい形に変える。

### Portfolio Value

案件選別、応募文作成、実績化までの一連の流れを作れます。
受託活動の前後をまとめて改善するツール群として見せられます。


## Client CRM

応募した案件、受注案件、クライアント情報を一覧で管理するブラウザツールです。
ステータス変更と Markdown 出力に対応し、次にやることを見失わないようにします。

### What It Solves

- 応募中、面談、契約、作業中などの状態をまとめて見たい。
- 次にやることを案件ごとに残したい。
- 日報やNotionに貼れる形で案件一覧を出力したい。

### Portfolio Value

案件選別、応募文作成、進行管理、実績化までの流れをつなぎます。
受託活動を実務ツール群として見せられる中核ツールです。


## Work Log Builder

日々の作業内容、成果、気づき、次アクションを短く記録し、実績化候補だけを抽出するブラウザツールです。

### What It Solves

- 毎日の作業を忘れずに残したい。
- あとからポートフォリオ化できる作業だけ見つけたい。
- 作業内容を日報やNotionに貼れる Markdown にしたい。

### Portfolio Value

作業ログから Portfolio Case Builder へつなげることで、日々の実務を見せられる成果物へ変換できます。
引退OSの「記録から資産化する」流れを補強するツールです。
