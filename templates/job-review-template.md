# Job Review Template

案件や企画を判断するための記録テンプレートです。

## Basic Info

- 案件名:
- 評価日:
- 種別: 受託 / ポートフォリオ / 自社事業 / 共通
- 状態: 応募 / 保留 / 見送り

## Summary

何をする案件か、1から3行で書く。

## Scores

- 売上性: /5
- 再利用性: /5
- 実行速度: /5
- 証明価値: /5
- 戦略性: /5
- リスク低さ: /5
- 作業負荷の低さ: /5

合計: /35

## Reason

なぜその判断にしたかを書く。

## Next Action

次にやることを1つだけ書く。

## Notes

気になる点、条件、あとで見返したい情報を書く。
