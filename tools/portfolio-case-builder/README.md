# Portfolio Case Builder

完了した案件や制作物を、ポートフォリオに載せやすい実績紹介文へ変換するブラウザツールです。
依存ライブラリなしで、`index.html` を開くだけで使えます。

## Purpose

受託案件や自作ツールを「やったこと」で終わらせず、見せられる実績として整理します。
課題、対応内容、工夫、成果、次に使える学びを Markdown で出力します。

## How To Use

1. `index.html` をブラウザで開く。
2. 制作物名、種類、課題、対応内容、工夫、成果、使用技術、公開可否メモを入力する。
3. 右側の実績紹介文を確認する。
4. `Markdown をコピー` を押して、GitHub README、Notion、ポートフォリオページへ貼る。

## Inputs

- 制作物名
- 種類
- 課題
- 対応内容
- 工夫した点
- 成果
- 使用技術
- 公開可否メモ

## Output

- ポートフォリオ掲載用の Markdown 実績紹介文

## Design Notes

- HTML 単体で動かす。
- 外部 API や認証は使わない。
- 入力項目は JavaScript の `fields` 配列で管理し、追加しやすくする。
- 個人名、顧客名、契約条件などを公開しない前提で書けるようにする。
