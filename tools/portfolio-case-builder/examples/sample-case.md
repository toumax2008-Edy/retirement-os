# Opportunity Scorecard

## Overview

案件や企画を同じ基準で採点し、応募 / 保留 / 見送りを判断するブラウザツールです。
HTML ファイルを開くだけで動作し、判断結果を Markdown としてコピーできます。

## Problem

受託案件や自社事業アイデアを、気分や緊急度だけで選んでしまう課題がありました。
判断理由を残せないと、あとから振り返って改善することも難しくなります。

## Solution

売上性、再利用性、実行速度、証明価値、戦略性、リスク低さ、作業負荷の低さを採点し、応募 / 保留 / 見送りを判断できるようにしました。

## What I Focused On

- 入力から判定までを1画面で完結させること。
- Markdown で記録を残せること。
- 外部ライブラリなしで誰でも開けること。
- GitHub 公開時に個人情報や認証情報を含まないこと。

## Result

案件選別の判断基準を統一し、応募文作成ツールへつなげられる実務フローを作れました。
