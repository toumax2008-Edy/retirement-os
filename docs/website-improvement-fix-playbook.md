# Website Improvement Fix Playbook

小規模事業者向け「サイト改善ミニ支援」の基本設計書です。

Website Improvement Analyzer v1.5 がサイトを診断するためのツールだとすると、このPlaybookは診断後に「何を直せるか」「何を受けないか」「どのように提案するか」を決める改善レシピ集です。

将来作成する Website Fix Plan Builder の元資料として使います。

## 基本方針

なおぴーが最初に提供するのは、本格Web制作ではなく、小規模事業者向けのサイト改善ミニ支援です。

中心にする対応範囲は、文章、導線、見せ方、簡易画像、修正指示書です。

- キャッチコピー改善
- サービス説明文改善
- CTA文言改善
- 問い合わせ導線の整理
- ファーストビュー構成案
- 信頼材料の見せ方改善
- FAQ作成
- CanvaやAIを使った簡易画像改善
- 修正指示書の作成

最初から大きく作り替えず、今あるサイトの良い材料を活かして、問い合わせや相談につながる形へ整えることを目的にします。

## サイト改善ミニ支援の対応メニュー

### 無料簡易診断

- 価格目安: 0円
- 内容: サイトを見て、良い点、改善点、優先順位を簡単に整理する。
- 納品物: 簡易診断メモ、改善候補3点、次の提案。
- 目的: 信頼づくりと有料支援への入口にする。

### ミニ改善案作成

- 価格目安: 3,000〜5,000円
- 内容: ファーストビュー、CTA、サービス説明、信頼材料の改善案をまとめる。
- 納品物: MarkdownまたはPDF化できる改善案、短い提案文、改善優先度。
- 目的: 小さく試せる提案商品にする。

### 修正指示書作成

- 価格目安: 10,000円前後
- 内容: Web担当者や制作会社に渡せる修正指示書を作る。
- 納品物: 修正箇所一覧、修正前後の文章案、優先順位、確認事項。
- 目的: 自分で実装しない案件でも価値を出せるようにする。

### 文章・CTA改善パック

- 価格目安: 15,000〜30,000円
- 内容: キャッチコピー、サービス説明文、CTA文言、FAQを改善する。
- 納品物: 修正文案、CTA案、FAQ案、掲載順の提案。
- 目的: 実装前の文章改善だけで成果を出しやすくする。

### Canva画像/バナー改善パック

- 価格目安: 15,000〜30,000円
- 内容: 簡易バナー、図解、SNS用画像、ファーストビュー用画像案を作る。
- 納品物: Canvaデザイン、画像案、差し替え指示。
- 目的: 見た目の印象を低コストで改善する。

### 実装サポート相談

- 価格目安: 別途相談
- 内容: HTMLの軽微な修正、既存ページへの文章反映、修正手順の相談を行う。
- 納品物: 修正案、実装メモ、作業ログ。
- 注意: WordPressの深い技術改修、サーバー作業、決済や予約システムは原則受けない。

## 価格の目安

価格は固定ではなく、実績作りと相手の規模に応じた目安です。

| メニュー | 価格目安 | 向いている相手 |
| --- | ---: | --- |
| 無料簡易診断 | 0円 | 初回相談、知人紹介、営業入口 |
| ミニ改善案 | 3,000〜5,000円 | まず小さく相談したい事業者 |
| 修正指示書込み | 10,000円前後 | 制作会社やWeb担当者に渡したい事業者 |
| 文章/画像改善込み | 15,000〜30,000円 | 見せ方と文章をまとめて直したい事業者 |
| 実装反映込み | 別途相談 | 既存サイトに軽微な反映まで希望する事業者 |

価格を出す時は、最初に「今回は小さく改善する範囲です」と伝えます。大規模制作ではなく、問い合わせにつながる見せ方を整える支援として説明します。

## 受けてよい案件

- 小規模事業者の既存サイト改善
- トップページの見せ方改善
- 問い合わせ導線の改善
- サービス説明文の改善
- CTA文言の改善
- 実績・口コミ・会社情報の見せ方改善
- CanvaやAIを使った簡易画像改善
- Web担当者や制作会社へ渡す修正指示書作成
- FAQやよくある不安への回答文の作成
- 地域名やサービス名を自然に入れた文章改善

## 受けない方がよい案件

- サーバー移転
- 大規模リニューアル
- 決済機能追加
- 予約システム構築
- WordPressの深い技術改修
- 本格SEO運用
- 広告運用代行
- 高度なアクセス解析
- ECサイトの大規模改修
- 法務、医療、金融など専門判断が必要な表現監修

受けない案件は、無理に断るのではなく「専門業者に任せた方が安全です」と伝えます。そのうえで、文章整理や修正指示書だけなら対応できる場合があります。

## クライアント確認事項

改善作業に入る前に、次を確認します。

- サイトの管理画面に入れるか。
- 誰がサイトを編集できるか。
- 問い合わせは電話、LINE、フォームのどれを優先するか。
- 掲載できる実績やお客様の声はあるか。
- 写真素材はあるか。
- 料金やサービス内容を公開してよいか。
- 修正反映まで希望するか、指示書だけでよいか。
- 既存の制作会社やWeb担当者がいるか。
- 使ってはいけない表現や掲載NGの情報はあるか。
- いつまでに改善したいか。
- 成果として何を重視するか。問い合わせ、予約、資料請求、LINE追加など。

## 改善レシピ 1: SEO

### よくある問題

- ページタイトルが抽象的で、何のサービスか分かりにくい。
- 見出しに地域名やサービス名が入っていない。
- 事業内容は書いてあるが、検索されそうな言葉になっていない。
- サービスごとの説明が短く、検索にもユーザー理解にも弱い。
- FAQがなく、相談前の不安に答えられていない。

### なおぴーが対応できる改善

- タイトル案と見出し案を作る。
- 地域名、サービス名、悩みの言葉を自然に入れる。
- サービス説明文を分かりやすく書き直す。
- FAQ案を作る。
- 既存ページに追加する文章案を作る。

### ChatGPT / Codex / Antigravity / Canva / AIに作らせるとよいもの

- ChatGPT: タイトル案、見出し案、サービス説明文、FAQ案。
- Codex: HTML上の見出し構造の確認、修正指示書化。
- Antigravity: SEO改善の判断メモ、ナレッジ整理。
- Canva: SEOそのものではなく、説明を補助する図解。
- AI: 検索されそうな質問文やFAQ候補。

### 専門業者に任せた方がよいもの

- 本格的なSEO戦略。
- 競合調査やキーワードボリューム調査。
- テクニカルSEOの大規模改善。
- 構造化データやサイト全体設計の専門対応。

### 営業での伝え方

「検索順位を保証するものではありません。ただ、今のサイトにある良い情報を、検索する人と初めて見る人の両方に伝わりやすい文章へ整えることはできます。」

### 修正前後の例文

修正前:

```text
地域に根ざしたサービスを提供しています。
```

修正後:

```text
沖縄で賃貸管理・相続対策・土地活用に悩む方を、初回相談からサポートします。
```

### 小規模事業者向けの提案文テンプレート

```text
現在のサイトにはサービス内容が掲載されていますが、検索する人が使いそうな言葉が見出しや説明文にもう少し入ると、内容が伝わりやすくなります。まずは地域名、サービス名、よくある悩みを自然に入れた見出しと説明文へ整えるのがおすすめです。
```

### 納品物の例

- タイトル案 3案
- 見出し案 5案
- サービス説明文
- FAQ 5〜10問
- 修正指示書

### 注意点

- 検索順位の保証はしない。
- 誇大表現は避ける。
- 専門分野の表現はクライアント確認を必須にする。

## 改善レシピ 2: CTA

### よくある問題

- 問い合わせボタンが目立たない。
- ボタン文言が「送信」「詳細」など抽象的。
- 電話、LINE、フォームの優先順位が分からない。
- ページ下部まで行かないと問い合わせできない。
- 相談前の不安を減らす説明がCTA周辺にない。

### なおぴーが対応できる改善

- CTA文言を具体化する。
- 問い合わせ導線の優先順位を整理する。
- ファーストビュー付近に置くCTA案を作る。
- CTA周辺に添える安心文を作る。
- 電話、LINE、フォームの使い分け文を作る。

### ChatGPT / Codex / Antigravity / Canva / AIに作らせるとよいもの

- ChatGPT: CTA文言、安心文、営業文。
- Codex: ボタン配置の修正案、HTML修正指示。
- Antigravity: 導線設計の整理。
- Canva: CTA用バナー、LINE誘導画像。
- AI: A/B比較用の文言案。

### 専門業者に任せた方がよいもの

- 予約システム構築。
- フォームの高度な改修。
- CRM連携。
- 決済や会員登録を伴う導線設計。

### 営業での伝え方

「今のサイトを大きく作り替えなくても、問い合わせボタンの見せ方と言葉を整えるだけで、次に何をすればよいかが分かりやすくなります。」

### 修正前後の例文

修正前:

```text
お問い合わせ
```

修正後:

```text
無料で相談する
```

修正前:

```text
詳細はこちら
```

修正後:

```text
管理について相談する
```

### 小規模事業者向けの提案文テンプレート

```text
サイトを見た方が迷わず行動できるように、問い合わせボタンの文言と配置を整理するのがおすすめです。特に「無料相談」「LINEで相談」「見積もりを依頼」など、次の行動が分かる言葉にすると、相談へのハードルを下げられます。
```

### 納品物の例

- CTA文言 5〜10案
- CTA配置案
- CTA周辺の安心文
- 問い合わせ導線の優先順位メモ
- 修正指示書

### 注意点

- 実際にフォームやLINEが動くかは必ず確認する。
- 電話を優先するか、フォームを優先するかをクライアントに確認する。
- 煽りすぎる文言は避ける。

## 改善レシピ 3: スマホ対応

### よくある問題

- スマホで文字が小さい。
- ボタンが押しにくい。
- 画像が大きく、重要な情報が下に流れる。
- 余白が少なく、読みづらい。
- ファーストビューで何のサイトか分かりにくい。

### なおぴーが対応できる改善

- スマホで先に見せる情報の順番を整理する。
- 見出しとCTAを短くする。
- 画像、説明文、CTAの優先順位を提案する。
- スマホ表示用の修正指示書を作る。
- 文字量を減らしたスマホ向け文章案を作る。

### ChatGPT / Codex / Antigravity / Canva / AIに作らせるとよいもの

- ChatGPT: スマホ向け短文、見出し、CTA文言。
- Codex: レスポンシブ修正案、HTML/CSSの軽微な確認。
- Antigravity: スマホでの見せ方改善案。
- Canva: 縦長バナー、スマホ向け図解。
- AI: 情報の優先順位案。

### 専門業者に任せた方がよいもの

- サイト全体のレスポンシブ改修。
- 複雑なレイアウト崩れの修正。
- WordPressテーマの深い修正。
- アプリや予約システムを含む表示調整。

### 営業での伝え方

「今は多くの人がスマホでサイトを見ます。まずスマホで見た時に、何のサービスか、何を相談できるか、どこから問い合わせるかがすぐ分かる状態に整えるのが効果的です。」

### 修正前後の例文

修正前:

```text
当社はお客様に寄り添ったサービスを提供しています。
```

修正後:

```text
外壁塗装・水回りリフォームの無料相談はこちら
```

### 小規模事業者向けの提案文テンプレート

```text
スマホで見た時に重要な情報が下に流れているため、最初に見える範囲で「何を相談できるか」と「どう問い合わせるか」が分かるように整理するのがおすすめです。文章を短くし、CTAを上部に置くことで迷いを減らせます。
```

### 納品物の例

- スマホ表示の改善メモ
- 情報の並び替え案
- 短い見出し案
- スマホ向けCTA文言
- 修正指示書

### 注意点

- 実機確認ができる場合はスマホで必ず見る。
- 見た目の修正だけでなく、問い合わせ導線の優先順位を確認する。
- 実装が深くなる場合は専門業者に任せる。

## 改善レシピ 4: ファーストビュー

### よくある問題

- 最初の画面で何の会社か分からない。
- 誰向けのサービスか分からない。
- 強みが抽象的。
- 問い合わせ導線が見えない。
- きれいな画像はあるが、行動につながっていない。

### なおぴーが対応できる改善

- キャッチコピーを具体化する。
- サブコピーを作る。
- ファーストビューの構成案を作る。
- CTA文言を作る。
- 画像やバナーの差し替え案を作る。

### ChatGPT / Codex / Antigravity / Canva / AIに作らせるとよいもの

- ChatGPT: キャッチコピー、サブコピー、CTA文言。
- Codex: HTML構成案、配置メモ、修正指示書。
- Antigravity: 見せ方やバナー方向性の整理。
- Canva: ファーストビュー用バナー、簡易メイン画像。
- AI: コピー案の比較、訴求軸の整理。

### 専門業者に任せた方がよいもの

- デザイン全体の作り替え。
- ブランド設計。
- 写真撮影。
- アニメーションや高度なビジュアル制作。

### 営業での伝え方

「最初に見える画面で、誰向けに何を提供しているかが伝わると、サイトを見た人が読み進めやすくなります。まずはキャッチコピーとCTAを整えるだけでも印象が変わります。」

### 修正前後の例文

修正前:

```text
人と人をつなぐ
```

修正後:

```text
沖縄の賃貸管理・相続・土地活用を初回相談からサポート
```

### 小規模事業者向けの提案文テンプレート

```text
現在のトップ部分は雰囲気は伝わりますが、初めて見た方に「何を相談できるか」がもう少し具体的に伝わると、問い合わせにつながりやすくなります。まずはキャッチコピー、サブコピー、問い合わせボタンを整理するのがおすすめです。
```

### 納品物の例

- キャッチコピー 5案
- サブコピー 3案
- CTA文言 5案
- ファーストビュー構成案
- Canvaバナー案

### 注意点

- かっこよさより、誰に何を提供するかを優先する。
- 誇張しすぎない。
- 業種の専門表現はクライアントに確認する。

## 改善レシピ 5: 信頼材料

### よくある問題

- 実績はあるが目立たない。
- お客様の声がない。
- 会社情報が分かりにくい。
- 料金や対応範囲が曖昧。
- 初めて相談する人の不安に答えられていない。

### なおぴーが対応できる改善

- 実績や事例の見せ方を整理する。
- お客様の声の掲載フォーマットを作る。
- 会社情報や許認可情報の見せ方を改善する。
- 料金や対応範囲の説明文を作る。
- FAQを作る。

### ChatGPT / Codex / Antigravity / Canva / AIに作らせるとよいもの

- ChatGPT: 実績紹介文、お客様の声テンプレート、FAQ。
- Codex: 実績セクションのHTML案、修正指示書。
- Antigravity: 信頼材料の整理、掲載順の判断。
- Canva: 実績紹介画像、口コミカード、図解。
- AI: 相談前の不安リストと回答案。

### 専門業者に任せた方がよいもの

- 法的チェックが必要な表現。
- 医療、金融、士業などの専門監修。
- 大規模な導入事例ページ制作。
- 写真撮影や本格的な取材記事制作。

### 営業での伝え方

「すでにある実績や会社情報を、初めて見る人が安心できる形に並べ替えるだけでも、相談前の不安を減らせます。」

### 修正前後の例文

```text
多くのお客様にご利用いただいています。
```

修正後:

```text
地域の賃貸オーナー様から、管理相談や相続に関するご相談を継続的にいただいています。
```

### 小規模事業者向けの提案文テンプレート

```text
サイト内に信頼材料はありますが、初めて見る方が安心できる形に整理すると、問い合わせ前の不安を減らせます。実績、会社情報、お客様の声、よくある質問を分かりやすく見せるのがおすすめです。
```

### 納品物の例

- 実績紹介文
- お客様の声テンプレート
- FAQ案
- 会社情報セクション改善案
- 修正指示書

### 注意点

- 実績や口コミは事実確認を必ず行う。
- 顧客名や写真を出す場合は許可を確認する。
- 料金を出す場合は公開可否を確認する。

## 改善レシピ 6: 表示速度

### よくある問題

- 画像が重い。
- ファーストビューの読み込みが遅い。
- 不要な画像や装飾が多い。
- スマホで表示に時間がかかる。
- 速度の問題なのか、見せ方の問題なのか切り分けられていない。

### なおぴーが対応できる改善

- 体感速度の確認メモを作る。
- 重そうな画像や不要そうな画像をリストアップする。
- 画像圧縮や差し替えの指示書を作る。
- ファーストビューで先に見せる情報を整理する。
- CanvaやAIで軽めの代替画像案を作る。

### ChatGPT / Codex / Antigravity / Canva / AIに作らせるとよいもの

- ChatGPT: 速度改善の説明文、クライアント向け注意書き。
- Codex: HTML上の画像位置や読み込み箇所の確認、軽微な修正案。
- Antigravity: 画像の見せ方改善案。
- Canva: 軽めの代替バナー、圧縮しやすい画像。
- AI: 画像差し替え案、説明図。

### 専門業者に任せた方がよいもの

- サーバー設定。
- CDN設定。
- JavaScriptの高度な最適化。
- WordPressプラグインやテーマの深い速度改善。
- PageSpeed Insightsの本格対応。

### 営業での伝え方

「速度改善は専門的な領域もありますが、まずは重そうな画像や最初に見せる内容を整理するだけでも、体感の見やすさは改善できます。」

### 修正前後の例文

修正前:

```text
大きな画像を複数枚並べて、サービス説明が下に流れている。
```

修正後:

```text
最初に見せる画像を1枚に絞り、見出し、説明文、問い合わせボタンを先に表示する。
```

### 小規模事業者向けの提案文テンプレート

```text
表示速度はサーバーやシステム側の専門対応が必要な場合もありますが、まずは画像の量や表示順を見直すことで、スマホで見た時の体感を改善できる可能性があります。今回は画像の整理とファーストビューの見せ方から確認するのがおすすめです。
```

### 納品物の例

- 画像見直しリスト
- 圧縮・差し替え指示
- ファーストビュー表示順の改善案
- 軽量バナー案
- 専門業者へ渡す相談メモ

### 注意点

- 速度スコア改善を保証しない。
- サーバーやCMSの深い設定は受けない。
- 実装前にバックアップや編集権限を確認する。

## 営業時の基本トーク

```text
本格的なWeb制作やシステム改修ではなく、今あるサイトの良い材料を活かして、問い合わせにつながりやすい見せ方へ整えるミニ改善支援です。

まずは無料または低価格で簡易診断を行い、ファーストビュー、CTA、サービス説明、信頼材料の見せ方を確認します。

必要であれば、修正指示書や文章改善案まで作成できます。サーバー移転や予約システム構築などの専門領域は、必要に応じて専門業者への相談をおすすめします。
```
