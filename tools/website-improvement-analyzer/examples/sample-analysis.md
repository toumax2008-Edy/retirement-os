# サイト改善診断レポート

## サイト概要

- サイト名: サンプル整体院 LP
- URL: https://example.com/sample-lp
- 業種: 整体 / 予約サービス
- サイト種別: LP
- ターゲット: 肩こりや腰痛に悩む30代から50代の地域住民

## 良い点

- ファーストビューで整体院の雰囲気が伝わる。
- 料金と施術内容がまとまっている。
- 予約ボタンがページ内に複数ある。

## 改善点

- 初回訪問者にとって、どの症状に強い整体院なのかが少し分かりにくい。
- 予約ボタンの文言が弱く、押した後の流れが見えにくい。
- スマホ表示では重要な情報が下に流れやすい。

## 優先順位

高

## AI活用の提案

- 症状別の説明文をAIで複数案作成する。
- 予約ボタン周辺の文言を、初回向けに改善する。
- よくある質問をAIで整理し、問い合わせ前の不安を減らす。

## 次のアクション

1. ファーストビューの見出しを症状別に改善する。
2. 予約ボタンの文言を「初回相談を予約する」に変更する。
3. スマホ表示で料金、施術内容、予約導線を上部に近づける。

## 提案書のたたき台

サンプル整体院 LPを拝見しました。全体として店舗の雰囲気は伝わっていますが、初回訪問者にとって「自分の悩みに合う整体院か」が少し判断しにくい状態です。

まずはファーストビューの見出し、予約導線、スマホ表示の情報配置を改善することで、問い合わせ前の迷いを減らせる可能性があります。

AIを活用すれば、症状別の説明文、よくある質問、予約ボタン周辺の文言改善を短時間で複数案作成できます。まずは高優先度の改善として、見出しとCTAの見直しから進めることを提案します。
