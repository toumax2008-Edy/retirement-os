# サイト改善診断レポート

## サイト概要

- サイト名: サンプル整体院 LP
- URL: https://example.com/sample-lp
- 業種: 整体 / 予約サービス
- サイト種別: LP
- ターゲット: 肩こりや腰痛に悩む30代から50代の地域住民

## 総評

店舗の雰囲気は伝わっています。初めて訪問した人が、自分の悩みに合う整体院かを判断しやすくすると、予約につながりやすくなります。

## 強み

- 店舗の雰囲気が伝わる。
- 料金と施術内容がまとまっている。
- 予約ボタンが複数箇所にある。

## 良い点

- 清潔感がある。
- 初回利用者に必要な情報が掲載されている。
- 予約導線が用意されている。

## 改善点

- どの症状に強い整体院なのかを見出しで伝えたい。
- 予約ボタンの文言を具体的にしたい。
- スマホ表示で重要情報を上部に近づけたい。

## 優先順位

- 優先度: 高
- 理由: ファーストビューとCTAを整えるだけで、初回訪問者の迷いを減らせるため。

## 診断チェックリスト

### SEOチェック

- タイトルが分かりやすいか: 要確認
- 見出しが整理されているか: 改善必要
- 検索されそうな言葉が入っているか: 要確認
- 事業内容・サービス内容が明確か: OK

### CTAチェック

- 問い合わせボタンが分かりやすいか: OK
- 次に取ってほしい行動が明確か: 要確認
- ボタン文言が具体的か: 改善必要
- CTAがページ内に複数あるか: OK

### スマホ対応チェック

- スマホで文字が読みやすいか: OK
- ボタンが押しやすいか: 要確認
- 画像や文章が詰まりすぎていないか: 要確認
- 重要情報がすぐ分かるか: 改善必要

### ファーストビューチェック

- 最初の画面で何のサイトか分かるか: 要確認
- 誰向けのサービスか分かるか: 改善必要
- 強みや特徴が伝わるか: 改善必要
- 問い合わせ・購入などの導線が見えるか: 要確認

### 信頼材料チェック

- 実績・事例・お客様の声があるか: 要確認
- 会社情報・店舗情報が分かるか: OK
- 料金やサービス内容が分かりやすいか: OK
- 不安を減らす説明があるか: 要確認

### 表示速度チェック

- 表示が重く感じるか: 未確認
- 画像が多すぎないか: 要確認
- ファーストビューの読み込みが遅くないか: 未確認

## 改善優先度

### 最優先で直すべき点

- ファーストビューの見出しと予約ボタンの文言を整理する。

### 次に直すべき点

- スマホ表示で料金、施術内容、予約導線を上部に近づける。

### 余裕があれば直す点

- 症状別ページ、FAQ、お客様の声を追加する。

- 問い合わせ導線: 予約ボタンの文言を具体化したい。
- スマホ対応: 重要情報を上部に近づけたい。
