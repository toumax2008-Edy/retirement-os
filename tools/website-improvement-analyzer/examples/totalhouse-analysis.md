# 株式会社トータル・ハウスマネジメント サイト改善診断レポート

## サイト概要

- サイト名: 株式会社トータル・ハウスマネジメント
- URL: https://www.totalhouse-mgt.jp/
- 業種: 不動産管理 / 不動産コンサルティング
- サイト種別: ホームページ
- ターゲット: 沖縄県内で賃貸管理、不動産売買、相続対策、借地、土地活用について相談したい個人・法人

## 総評

信頼材料は十分にあります。対応業務の幅、管理物件の実績、会社情報が掲載されているため、初めて見る人にも実在感は伝わります。今後は、その信頼材料を問い合わせにつながる形に整理することが改善の中心になります。

## 強み

- 賃貸管理、不動産売買、相続対策、借地、土地活用まで幅広く対応できる。
- 管理物件や実績が掲載されている。
- 会社情報、連絡先、宅建業免許番号が確認できる。

## 良い点

- 賃貸管理、不動産売買、相続対策、借地、土地活用など対応範囲が広い。
- 管理物件や実績が掲載されており、実在感と信頼材料がある。
- 会社情報、所在地、連絡先、宅建業免許番号が掲載されている。
- 初回相談無料の案内があり、相談への心理的ハードルを下げられる。
- 沖縄県内の具体的な管理エリアや物件種別が見える。

## 改善点

- ファーストビューで「誰の、どんな悩みを解決する会社か」がやや伝わりにくい。
- 「人と人をつなぐ」は印象の良い言葉だが、検索や問い合わせ目的では少し抽象的。
- 問い合わせ導線が目立ちにくく、初めて訪問した人が次に何をすればよいか迷いやすい。
- 管理物件の実績はあるが、「どんな強みがある会社なのか」まで整理されていない。
- 賃貸管理、相続対策、借地相談、土地活用などの相談導線を分けると、ユーザーが相談しやすくなる。

## 診断メモ

- 第一印象: 信頼素材はあるが、問い合わせにつなげる導線が少し弱い。
- デザイン: 会社の雰囲気は伝わるが、見出しとCTAにもう少し強弱をつけたい。
- 情報の分かりやすさ: 対応業務は多いが、サービス別に整理すると初見の理解が早くなる。
- 問い合わせ導線: 無料相談や問い合わせボタンをファーストビュー近くで目立たせたい。
- スマホ対応: 重要情報と問い合わせ導線が下に流れやすい可能性があるため、上部に集約したい。
- 表示速度: 画像が多い場合は、画像圧縮と読み込み順の見直し余地がある。
- SEOの気づき: 「沖縄」「賃貸管理」「相続対策」「借地相談」「土地活用」などの検索語を、見出しや説明文に自然に入れる余地がある。

## 優先順位

高

理由: 信頼材料はすでに掲載されているため、ファーストビューの見出し、無料相談CTA、サービス別導線を整理すれば、比較的少ない修正で問い合わせ改善につながる可能性が高い。

## 提案書のたたき台

株式会社トータル・ハウスマネジメント様のサイトを拝見しました。

現在のサイトは、会社情報、対応業務、管理物件の実績が掲載されており、信頼材料は十分にあります。特に、賃貸管理、不動産売買、相続対策、借地、土地活用まで対応できる点は大きな強みです。

一方で、初めて訪問した人が「自分は何を相談できるのか」「まず何をすればいいのか」を判断する導線がやや弱い印象です。

まずはファーストビューの見出しを具体化し、無料相談ボタンを目立たせ、賃貸管理・相続・借地相談・土地活用ごとの導線を整理することで、問い合わせ率の改善が期待できます。

AIを活用すれば、サービス別の説明文、FAQ、管理実績の紹介文、問い合わせ導線の文言を短時間で複数案作成できます。

最初の改善としては、見出し、CTA、サービス別導線の整理から着手するのがおすすめです。
